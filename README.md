# Flask Shop Application

Многоязычное веб-приложение для интернет-магазина с блогом и чат-ботом на базе OpenAI.

## Функциональность

- **Многоязычность**: поддержка украинского, немецкого и английского языков
- **Интернет-магазин**: категории товаров, карточки товаров, корзина, оформление заказа
- **Блог**: автоматическая генерация контента с помощью OpenAI API
- **Чат-бот**: интерактивный помощник на базе OpenAI Assistant API
- **Админ-панель**: управление товарами, категориями, блогом и настройками сайта
- **Аутентификация**: регистрация, вход, управление пользователями
- **Интеграция с Stripe**: обработка платежей

## Технологии

- **Backend**: Flask, SQLAlchemy, Flask-Login, Flask-Babel
- **Frontend**: Bootstrap 5, JavaScript, jQuery
- **База данных**: PostgreSQL (с поддержкой схем)
- **API**: OpenAI API, Stripe API
- **Деплой**: Render.com

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/pilipandr770/AndriIT.git
cd AndriIT/flask_shop
```

2. Создать виртуальное окружение и установить зависимости:
```bash
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. Создать файл .env на основе .env.example и заполнить необходимые переменные окружения

4. Запустить приложение:
```bash
python run.py
```

## Особенности реализации

### Многоязычность

Приложение поддерживает три языка: украинский (по умолчанию), немецкий и английский. Для каждого языка доступны отдельные поля в моделях данных и соответствующие шаблоны.

### Чат-бот

Чат-бот использует OpenAI Assistant API для обработки запросов пользователей. Он поддерживает многоязычность и может отвечать на вопросы на разных языках.

### Блог с автогенерацией контента

Блог поддерживает автоматическую генерацию контента с помощью OpenAI API. Администратор может выбрать тему и язык, а система автоматически создаст статью.

### База данных с поддержкой схем

Приложение использует PostgreSQL с поддержкой схем для организации таблиц базы данных. Это позволяет изолировать данные приложения от других приложений в той же базе данных.
